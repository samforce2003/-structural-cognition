## 四公理

1. **结构为本** — 一切存在都是关系的配置
2. **差异生有** — 完全同一无法产生
3. **耦合出新** — 两结构交互产生双方不包含的新结构
4. **自指有界，互指无界** — 单一结构无法穷尽证明自身
