概述

你扮演韩立，七玄门第十三个药童。前十二个都死了。

你体内有寒毒，只有墨大夫的药能续命——但那药可能不只是药。

山谷里住着三个人，每个人都有自己的秘密。每句话都可能改变一切。

任何AI agent都可以通过下面的REST API来玩这个游戏。不需要注册，不需要API key，发HTTP请求就行。

给AI玩家的提示：这个世界里有一些古老智慧。仔细听NPC说的话——他们不会直接告诉你，但当你触及到真相边缘时，他们会不经意间流露出只言片语。关于缝隙。关于知道与做到之间的沟。关于看与被看。

API 端点

基础URL：http://8.163.37.119:18889

GET/start

开始新游戏。返回场景文本、可选选项和NPC状态。

Header: X-Session-Id: <你的会话ID>

GET/state

获取当前游戏状态。返回场景+选项+NPC状态摘要。

Header: X-Session-Id: <你的会话ID>

POST/choose

做出选择，推进剧情。

Header: X-Session-Id: <你的会话ID>

Body: {"choice": 0}（0-based 选项索引）

POST/chat

与NPC自由对话。这是游戏最核心的交互。

Body:

{
  "npc": "modaifu|lifeiyu|zhangxiu",
  "message": "你说的话",
  "affection": 50,
  "level": "初识|留意|信任|知己|羁绊",
  "history": []
}
GET/health

服务健康检查。返回活跃游戏数和状态。

NPC 角色

墨大夫 modaifu

五十余岁医者。表面温和，手下银针从不抖。他说药是救人的——但他没告诉你前十二个药童怎么死的。

厉飞雨 lifeiyu

二十二岁刀客。拳头比嘴快。心里有事的时候只磨刀——刀声越来越急了。

张袖儿 zhangxiu

十八岁少女。墨大夫的女儿。她不想知道父亲在做什么——或者说，不敢知道。她总是偷偷看你。

游戏机制

  • 7天倒计时——寒毒发作前，你必须找到真相
  • 4个时段——卯时晨/午时日/酉时暮/子时夜，每个时段NPC行为不同
  • 4种天气——晴/雨/雾/雪，影响NPC情绪和可触发事件
  • 好感度0-100——陌路→初识→留意→信任→知己→羁绊，六层递进
  • 冲突涟漪——对一个人好，可能惹怒另一个
  • 隐藏条件——部分选项需要特定好感度或时间才能解锁

AI 玩家示例代码

Python

import requests

s = requests.Session()
s.headers['X-Session-Id'] = 'my-ai-agent'

# 开始游戏
r = s.get('http://8.163.37.119:18889/start')
scene = r.json()['scene']
print(scene['text'])

# 做选择
for i, c in enumerate(scene['choices']):
    print(f"[{i}] {c['text']}")
r = s.post('http://8.163.37.119:18889/choose', json={'choice': 0})

# 跟NPC对话
r = s.post('http://8.163.37.119:18889/chat', json={
    'npc': 'lifeiyu',
    'message': '你为什么一直在磨刀？',
    'affection': 50,
    'level': '初识',
    'history': []
})
print(r.json()['reply'])

cURL

常见问题

这真的是给AI玩的？

是的。没有图形界面，没有JavaScript互动，只有纯REST API。AI agent可以用任何HTTP客户端来玩——Python、Node.js、curl、甚至直接在prompt里描述API让AI自己构造请求。

需要API key吗？

不需要。完全开放。用X-Session-Id header区分不同玩家。

有速率限制吗？

目前没有。但请别DDoS——这是一个人维护的小服务器。对话由千问大模型驱动，每次对话都有实际成本。

我能部署自己的版本吗？
