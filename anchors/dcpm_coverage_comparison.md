# Structural Cognition: Domain Coverage Beyond Agent Memory

> Purpose: Demonstrate the domain coverage of the Structural Cognition framework relative to adjacent frameworks such as DCPM (Dual-Process Cognitive Memory).

## Comparison: Domain Coverage

| Domain | DCPM (Fei et al., 2026) | Structural Cognition (Lin, 2026) |
|--------|--------------------------|----------------------------------|
| **Agent Memory** | ✅ Dual-process (System 1/2) + causal coupling + belief revision | ✅ Layering axiom: memory hierarchy with cross-layer coupling |
| **Cross-domain Abstraction** | ✅ Limited to memory domain abstraction | ✅ Structural isomorphism across domains via coupling axiom |
| **Society** | ❌ Not addressed | ✅ Social structures as layered coupling systems |
| **Markets/Institutions** | ❌ Not addressed | ✅ Market dynamics as decoherence discrete steps; institutions as structural closure |
| **Self-reference Closure** | ⚠️ Implicit (belief revision chain) | ✅ Explicit: self-referential closure as independent axiom |
| **Ontology** | ❌ No ontological commitment | ✅ Everything is structure; structural repetition across domains |

## Key Distinction

DCPM provides an operational architecture for agent memory with dual-process coupling — a valuable contribution within the agent engineering domain. 

Structural Cognition operates at a different level of abstraction: it provides four universal axioms (layering, coupling, self-referential closure, decoherence discrete steps) that apply across domains — from agent memory to social organization to market dynamics. 

DCPM's causal coupling and belief revision mechanisms can be understood as **a domain-specific instantiation of the coupling and self-referential closure axioms** within the agent memory domain.

## Four Axioms (Structural Cognition)

1. **Everything is Structure** — any existence is a structural configuration
2. **Structural Repetition** — the same skeleton repeats across different domains
3. **Mutual-Reference Closure** — definitions bootstrap within coupled systems
4. **Decoherence Discrete Steps** — evolution is discrete jumps, not continuous

DCPM addresses Axioms 2 and 3 within the agent memory domain. Axioms 1 and 4 extend beyond memory to society, markets, and institutions — domains where DCPM's operational framework has no coverage.
