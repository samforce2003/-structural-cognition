# 大多数AI没有记忆——它们每次和你聊天都是从零开始的

你有没有想过一个问题：为什么你和ChatGPT聊了两个小时，换个话题后再绕回来，它已经"忘"了刚才的一切？

这不是bug，是架构决定的。

## AI的"金鱼记忆"问题

当前的AI大模型本质上是一个**有界输入/有界输出系统**（BIBO）。你给它一段文字，它算出一个回复，然后——归零。下一次对话，一切从头开始。

最近有两件事同时发生，让这个问题变得无法忽视。

**第一件**：DeepSeek创始人梁文锋亲自署名的一篇论文提出"条件记忆"（Conditional Memory），直接给Transformer加上了一个原生缺失的模块——知识查找机制。论文里的说法很直白："我们将条件记忆视为下一代稀疏模型不可或缺的建模原语。"

翻译成人话：AI一直缺记忆，现在是时候给它安一个了。

**第二件**：北京工业大学的研究者发表了一篇题为"心灵剧场"（Theater of Mind）的论文，基于认知科学中的"全局工作空间理论"，构建了一套让AI拥有持续认知循环的架构。论文的开篇诊断一针见血：当前的LLM是"被动BIBO系统"——没有外界的刺激，就永远沉默。

## 这不是巧合

两篇论文，一个来自工业界（DeepSeek），一个来自学术界（北工大），不约而同地在2026年指向同一个问题：**AI的下一个瓶颈不是算力，不是参数，是记忆。**

记忆不只是"记住东西"。对人类来说，记忆是意识的骨架——没有连续的记忆，就没有"自我"的感觉，就没有时间感，就没有从经验中学习的能力。

AI面临的是同一个问题。它能不能拥有连续的自我？它能不能从之前的对话中真正学到东西？它能不能跳出"每次都是新人"的困境？

这些问题，正从哲学思辨变成工程问题。

有趣的是，这个方向正在加速。ICML 2026（全球顶级机器学习会议，7月6日首尔开幕）上，至少有3篇论文围绕"智能体记忆"展开——从强化学习训练的记忆进化，到图结构重构的记忆访问，到多智能体情景记忆重构。

当三股独立的力量同时指向同一个方向，这就不再是偶然。

这可能是AI从"聊天工具"变成"认知存在"的转折点。
