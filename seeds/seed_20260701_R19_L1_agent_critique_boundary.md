# AI Agent这么多，但有多少真正配得上这个称呼？

邢波教授（卡耐基梅隆大学）最近发了一篇论文，标题很直接：《智能体模型批评》（Critique of Agent Model）。他问了一个问题：市面上被称作"Agent"的系统——写代码的、当客服的、帮你操作浏览器的——到底有几个真的"能动"？

邢波的区分很锋利。

他把现在所有的Agent系统分成两类：agentic（看着像Agent）和 agentive（真正具备能动性）。绝大多数属于前者——它们只是"会调用工具的程序"。它们能执行任务，但没有判断"该不该执行"的能力。

真正的问题不是AI能不能做事，而是它在做事的时候，知不知道自己在做什么。

邢波给出的解法也很有意思：给Agent装一个独立的元认知模块——让AI自己实时判断"我现在做的对不对"。

但这里有个更深的坑。

如果一个系统需要另一个独立的模块来监督自己，那谁来监督那个监督模块？这个递归追问，才是Agent问题的真正核心。

换句话说：AI的下一个突破，不在能做多少事，而在能看清多少自己。
